# python与java的区别

|   对比项   |           Python           |                 java                  |
| :--------: | :------------------------: | :-----------------------------------: |
|    类型    |          动态语言          |               静态语言                |
|  类型声明  |           不需要           |                 需要                  |
|    效率    |             低             |                  高                   |
|  适用场景  |          小型开发          |               软件工程                |
|     数     |  整型 长整数 浮点数 复数   | char short byte int long float double |
|   字符串   |         无char类型         |              char string              |
| 多行字符串 |           /表示            |                 +表示                 |
| 操作符表示 | ** 幂运算 //整除 ~按位翻转 |                                       |
| 序列化表示 |        str() repr()        |              toString()               |
