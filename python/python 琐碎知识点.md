# python 琐碎知识点

#### 函数中参数 * 与 ** 的差别

|               *                |                  **                  |
| :----------------------------: | :----------------------------------: |
|        接受参数为 元组         |           接受参数为 字典            |
| `def fun(*args )=>args(1,2,3)` | `def fun(**args)`=>args{a:1,b:2,c:3} |
