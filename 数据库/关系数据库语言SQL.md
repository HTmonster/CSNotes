# 关系数据库语言SQL

#### SQL功能特点

##### 功能

| SQL功能          | 核心动词                 |
| ---------------- | ------------------------ |
| 数据查询         | SELECT                   |
| 数据定义   (DDL) | CREATE,   DROP, ALTER    |
| 数据控制         | INSERT,   UPDATE, DELETE |
| 数据操纵   (DML) | GRANT,   REVOKE          |

##### 与一般关系模型的对照

| 一般关系模型 | SQL       |
| ------------ | --------- |
| 内模式       | 存储文件  |
| 模式         | 表 table  |
| 外模式       | 视图 view |
| 元组         | 行 row    |
| 属性         | 列 column |
