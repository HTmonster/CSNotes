# Linux重要目录详解

#### CentOS 7的根目录

```shell
- bin           #可执行二进制文件目录
- boot          #linux启动时候用的文件
- dev           #设备文件
- etc           #系统配置文件
- home          #系统默认 用户 家目录
- lib           #函数库
- lib64
- media
- mnt           #光盘挂载点
- opt           #主机额外安装软件 KDE煮面等
- proc          #系统核心，外部设备 网络状态  数据存放在内存中
- root          #系统 管理员 家目录
- run
- sbin          #系统管理员使用的可执行命令  例如fdisk shutdown
- srv           #服务启动之后访问的数据目录
- sys
- tmp           #临时文件
- usr           #应用程序存放目录
- var           #系统执行过程中变换的文件 如日志
```
